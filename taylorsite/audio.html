Audio

Billy Bob's Used Cars

I made a radio commercial for this project. The software I used was Audacity. I imported the voiceover and edited it to fit the script, then I added the music file, trimmed it to fit the length of the voice over and added a fade out to the end. I also found a car horn sound effect online that I added to the beginning. I think I did a pretty good job with this project. One thing that was hard was to find all of the parts of the script and picking the best take for each line.

Monster Trucks Rally

​For this project I was required to edit an audio recording for a commercial for a monster truck rally. I imported the main audio into Audacity and edited it according to the directions. I had to separate each line into different tracks so I could add effects to them individually. One thing that I did that made this project easier was that I named each individual track according to what line is spoken on it so I can easily know which track each line was on. I think I did pretty good on this project, It had all of the required effects and no words were cut off or anything. One thing that was hard for this project was learning to use the different effects. I remember for the part that I had to change the pitch I clicked apply and then tested it then edited it again, and it ended up making it way too deep, so I had to use the undo button to go back to before I edited that portion and make sure I didn’t click apply until I was completely happy with the sound of it.

Scripted Commercial

​For this project I had to record and edit a commercial for a company called Podcast Princess Productions. First, I recorded my audio. I had to record it several times to get it where I wanted, but I finally got it. The editing was fairly simple, I just had to make it where there was no background audio in the recording, and add music and a sound effect. the paper said to add a podcast intro and I was looking on this website and they all had this beep at the start to signal the person to start talking, so I decided to add that, and I found some good music that matched what it asked for, and I applied a short fade in and fade out.

School Function Ad

​For this project we had to write, record, and edit a commercial for a real or fictional event occurring at our school. I chose a movie event happening on the football field at BHP so people can go and be able to space out enough to be safe. I wrote the script using the things we learned about radio ads, then I recorded it and found music and a sound effect that fit with the theme then I edited it all together. I think I did okay on this project. I don’t think it’s the best I’ve done though. One thing that was hard was thinking of music that would fit well with the theme and mood of the commercial.

Mad World Ad

For this project I had to create a commercial for Mad World. I started out by listening to radio commercials made for other haunted houses. Then I went to the Mad World website to get some more information for the commercial. Then I wrote my script and recorded it. This one definitely took the most takes out of all of the projects. Then I edited it and I found some music and a sound effect that I liked and added those into my project. Overall I think I did pretty good for this project, but I definitely could’ve been better. One thing that was hard was trying to make it sound like other radio commercials for haunted houses, I tried adding effects and stuff to my voice, but it didn’t really work out that well, so I just did without the effects. ​

'I Believe' Essay

​For this project I had to write and record an essay about something I believe in. I wrote about how I believe that everyone should be treated equally no matter what. I started out by writing my essay and writing down everything I wanted to say about the topic. Then I recorded my audio. I tried something different for the recording for this project. I just recorded it all in one take and when I messed up I just started the line over. I also ended up recording it twice, so I could take the best of each line from each take. I edited this in a very similar way that I edited the first project we did. I made a separate track and cut all the things that I wanted to put in the final project and put them in the track at the bottom. I then found some music that fit the theme. I think I did okay on this project. The hardest thing was finding music that fit the mood of the essay, and I feel like I could’ve done a better job on that.
