Install a Virtual Machine in VirtualBox

Getting VirtualBox

First you will need to download VirtualBox from https://www.virtualbox.org/ Currently the latest version is 6.1. So, click the big button Download VirtualBox 6.1.

After you have clicked that button select the appropriate platform under VirtualBox 6.1.18 platform packages for me that will be Windows hosts

A file will download do not open it just yet. Next, we need to get the extension pack, just scroll down a bit until you see VirtualBox 6.1.18 Oracle VM VirtualBox Extension Pack and click All supported platforms.

You should now have 2 files downloaded. VirtualBox-6.1.18-142142-Win.exe and Oracle_VM_VirtualBox_Extension_Pack-6.1.18.vbox-extpack

Now just run the VirtualBox-6.1.18-142142-Win.exe and wait for it to load.

Once it loads just follow the on-screen instructions to install VirtualBox

After you have installed VirtualBox you will then need to install the extension pack. There are 2 ways you can install the extension pack one way is you can just run the file Oracle_VM_VirtualBox_Extension_Pack-6.1.18.vbox-extpack (double click) and the other way is to go to the File Menu then click Preferences a new window will popup. Then you will click the Extensions tab and click the green plus button.

Then just locate you VirtualBox extension pack file click it and click open.

Then you will need to just click Install for me it says Reinstall because I already have it installed.

Scroll down and click I Accept for the license agreement

The extension pack should now be installed.

You can check by looking in Extensions tab in the preferences window.

Thats it for installing VirtualBox

Creating a Virtual Machine

In the main VirtualBox Window click New do not click Add

A window will popup asking for the name, location, operating system type, and operating system version. Fill it out based on what you have. I recommend that you include the operating system name in the name filed because it will automatically determine the operating system based on that. If it does not pick the correct operating system and version, you will need to correct it. If you do not know the pick Other and Other/Unknown for 32 bit or Other/Unknown (64 bit) for 64 bit.

Next you will be asked how much memory (RAM) you want your virtual machine to have. If you are unsure just click Next.

Next you will be asked if you already have a virtual hard disk. If you are unsure just click Create.

It will then ask what hard disk file type you want. If you are unsure just click Next.

Then you will be asked if you want dynamically allocated or fixed size.

Next you will be asked where to store the virtual hard disk and what size you want it to be. If you are unsure just click Create.

Your virtual machine is technically created now, however there is now operating system.

Installing the Operating System

You will need to have an ISO file to install an operating system in your virtual machine. In this tutorial I will be using ReactOS which can be download from https://reactos.org/ I am using version 0.4.13 (ReactOS-0.4.13-release-14-g2494cfc.iso).

Now that you have created your virtual machine you are ready to install your operating system, but before you can install the operating system you will need to attach the iso file for your operating system.

Click settings

Click the Storage tab on the settings menu, then click the empty optical disk drive, then click the dropdown menu and click choose a disk file.



Locate you ISO file of the operating system that you will be installing in my case ReactOS my ISO file name is ReactOS-0.4.13-release-14-g2494cfc.iso so I will select that file and click Open.

After you click open the optical drive should now contain your ISO file. If so just click OK.

Now you are ready to start your virtual machine. All you need to do is click Start (The green arrow)

A window with your virtual machine should pop up

Now all thats left to do is install the operating system like you would on a physical machine. (Will vary by operating system).

Once you finish installing your operating system you will have a virtual machine.

You are now free to play around with your virtual machine or you can go to the settings and look at the other options for your virtual machine such as how many virtual processers you machine has and so on.

Have Fun!
